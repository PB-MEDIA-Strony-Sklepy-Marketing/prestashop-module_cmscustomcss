# prestashop-module_cmscustomcss
CMS Custom CSS — Moduł PrestaShop 9.0.3. Dodaje możliwość przypisania indywidualnego CSS do każdej strony CMS w PrestaShop. CSS jest przechowywany w dedykowanej tabeli bazy danych i wstrzykiwany do sekcji &lt;head> wyłącznie na tej konkretnej stronie CMS. Puste pole = zero output.
